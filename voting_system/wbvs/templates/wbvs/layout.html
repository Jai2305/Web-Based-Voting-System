FAQ
Q: What is web based voting system ?
Ans: A web based voting system is a software platform that allows groups to securely conduct votes and elections. High-quality online voting systems balance ballot security, accessibility, and the overall requirements of an organization's voting event.

Q: How will I benefit from the web based voting system?
Ans: Following are the benefits of Web Based Voting System -
  • Ease of operation. With User ID and password, you can login to e-Voting system
  • Sufficient time to vote till the end of voting cycle
  • Paperless mode of casting vote.

Q: How will I know if web based Voting website is secured?
Ans: We have ensured your voting security by implementing these steps -
  • Unique IDs for each candidate, each voter and each Polling Booth. This reduces chance for collisions
  • Hashed passwords for users. Not even Admin will be able to see the password of a particular user.
  • Only one time access for voters. No multiple votes will be counted
  • Voting Admin has the right to accept or reject a voter's voting request. It enables more secure and limited polling process
  • Not even Voting Admin will be able to know which candidate is recieving how much votes.

Q: How do I register for web based Voting facility?
Ans: You can register through the register button in the main page.

Q: Is there any charge for using this Voting system?
Ans: No. Currently, we do not levy any charge on the users for using the web based Voting system.

Q: Is there any User manual to understand the login and voting process ?
Ans: No, If you face any issue, just ask us. We are happy to help.

Q: Once I cast my vote on web based Voting System, can I modify my vote before the closing of Voting cycle?
Ans: No. Vote once casted will be considered final and cannot be modified.

Q: Who can use this voting system?
Ans: Any one who have registered their profile.

Q: Do I need special internet skills to cast my vote with web based voting system?
Ans: You don't need special internet skills to vote . This is because our system is created to be intuitive to use and you are provided with the information on every further step. The web based voting system operates like a website, so if you know how to navigate a website, then you'll be more than comfortable voting online. Once logged into our voting system with your access data, you'll be guided through the process of casting your vote.

Q: How can I create an online election with web based voting system?
To conduct your first online election with this system, all you need to do be a admin to conduct election. You'll then have the following options once in the admin options:
  • Generate one time Booth ID
  • Add Your Candidates with their respective names, description and photos
  • Finish Candidate Setup
  • Your Voting Is Live
  • Accept only recognizable IDs
  • You Can Reject those IDs which are not in your knowledge
  • View and declare the result

Q: How will the results be made available at the end of the voting period?
Ans: You can check your History Tab. Results will be availaible there after declaration from Voting Admin.
